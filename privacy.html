Privacy

Updated June 17, 2026.

This site collects basic analytics so I can understand visits and link clicks. It does not use cookies, advertising trackers, or browser fingerprinting.

What is collected

  • Page visits and link clicks.
  • The page URL, referrer, language, timezone, viewport, and screen size.
  • Approximate location and IP address from the server request.
  • User agent details sent by the browser.

Where it goes

Analytics events are sent through a serverless endpoint and forwarded to a private Discord webhook. The public website does not contain the webhook secret.

Opting out

The analytics script respects Global Privacy Control and browser Do Not Track signals.
